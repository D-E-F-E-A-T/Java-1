1ª Abra o NetBeans, abra o projeto da pasta "ManipulandoImagem" para poder
vizualizar o código. 
2ª Execute o programa (F6)
3ª com a janela do programa criado aberta, clique no menu arquivo
4º clique em "abrir imagem" selecione uma imagem para visualizar o resultado 
5ª Pronto. 
Caso queira pode clicar novamente em arquivo, e selecionar outra imagem! :)
